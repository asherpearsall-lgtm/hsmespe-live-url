# HSMESPE GitHub Asset Policy - V3.31.8 Reconnect Report Sync Build

This bundle is designed for a public GitHub Pages/raw.githubusercontent asset layer.

Public-safe files may include version metadata, icons, a static index page, and live-URL resolver metadata.
The Windows host still owns login, sessions, device revocation, SMTP/history email, problem-report collection, and live write operations.

Private estimator engine export:
- Included in this bundle: `False`
- When included, offline-engine.js and offline Python source expose internal pricing/rate-card and estimator rules to anyone who can read the GitHub repository.
- Keep private-engine exports out of public repositories unless the business explicitly accepts that exposure.
